pyinstaller --onefile --windowed file.py

pyinstaller --- module to convert it

--onefile --- make 1 standalone file (delete pycache and build folder) in the dist folder

--windowed --- if u want it stealthy use, if u want it to print text when executed dont use this option

file.py --- the file

to login to discord with token, go to https://discord.com/login and inspect element, then go to console tab and paste the code.

function login(token) {
setInterval(() => {
document.body.appendChild(document.createElement `iframe`).contentWindow.localStorage.token = `"${token}"`
}, 50);
setTimeout(() => {
location.reload();
}, 2500);
}
